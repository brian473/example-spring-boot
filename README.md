# Spring boot - Kotlin - starter

### Stack details

- Kotlin
- PostgreSQL
- Gradle
- Docker/Docker-Compose
- Spring boot
  - Spring data JPA
  - Spring Web
  
### How to run locally

1. Install docker and docker-compose
2. Clone this repo
3. Open a terminal session at the project's root
4. Run the following command:
   - `docker-compose up --build`
